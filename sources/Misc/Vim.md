Random tips and tricks to use Vim more efficiently

## Toggling line wrapping

To turn line wrapping on:

```
:set wrap
```

To turn line wrapping off:

```
:set nowrap
```

To toggle line wrapping on and off:

```
:set wrap!
```

## Toggling line number

To turn line wrapping on:

```
:set number
```

To turn line wrapping off:

```
:set nonumber
```

To toggle line wrapping on and off:

```
:set number!
```
